- **[FINDMAGSYM](https://findmagsym.streamlit.app/)** — automatic magnetic
  space-group symmetry parsing and spin-splitting-type (SST-1 to SST-7)
  classification for candidate compounds. Drop-in screening layer for
  high-throughput and AI-driven materials discovery pipelines.

- **[MAGIRREP](https://github.com/yuanlinding/RepresentationAnalysis/tree/main)**
  — Python tool that performs Bertaut representation analysis for
  magnetic structures from an `.mcif` file.
